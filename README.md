# Mangrove Impact Simulator

A polished React and Vite educational website where visitors use a slider to explore how planting mangroves can help transform a coastline.

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:5173`.

## Build

```bash
npm run build
```

## Notes

- The simulator uses CSS shapes, gradients, and simple inline SVG icons, so it does not require external image assets.
- Impact language is intentionally careful and educational. Actual outcomes depend on local species, age, density, site conditions, and ecosystem health.
